# B07 驾驶舱指标口径字典与计算规则设计

## 任务
Issue #9（B07）：统一定义驾驶舱指标口径与转化漏斗规则，且版本可追溯。

## 已确认范围
- 使用代码内版本字典（不落库）
- 本期不提供对外 API
- 固定漏斗阶段顺序：
  1) 登录激活
  2) 测评完成
  3) 报告生成
  4) 任务完成
  5) 活动参与
- 任务完成率口径：有任务学生中，至少完成 1 个任务的学生占比

## 指标口径定义
1. 测评完成率
- 分子：测评完成学生数
- 分母：登录激活学生数

2. 报告生成率
- 分子：报告生成学生数
- 分母：登录激活学生数

3. 任务完成率
- 分子：至少完成 1 个任务的学生数
- 分母：被分配任务的学生数

4. 活动参与率
- 分子：参与活动学生数
- 分母：活动覆盖学生数

5. 转化漏斗
- 阶段顺序固定，不允许动态重排
- 每阶段转化率 = 当前阶段人数 / 上一阶段人数（上一阶段为 0 时按 0 处理）

## 版本追溯方案
- 在 `src/modules/metrics/dictionary.ts` 中维护：
  - 当前版本号（如 `b07.v1`）
  - 版本变更日志（changelog）
  - 每个指标与漏斗定义标记所属版本
- 通过测试保证：
  - 关键指标定义完整
  - 漏斗阶段顺序固定
  - 版本与 changelog 一致可追溯
